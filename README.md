# Mesto (frontend + backend)

Сервис для обмена фотографиями, созданная в рамках учебы в Яндекс.Практикуме на курсе "Веб-разработчик".
Репозиторий включает в себя фронтенд и бэкенд части приложения.

## Технологии

**Frontend:**

- HTML5
- CSS3
- JavaScript [ES6]
- React

**Backend:**

- NodeJS
- ExpressJS
- MongoDB
